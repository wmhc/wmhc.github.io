如何用hexo部署自己的第一个个人博客

@(此操作基于macOs系统，windows，linux，同理)

[TOC]

  1. 进入root模式 sudo su
  2. 安装hexo需要的基本组件,直接下载安装就好node.js
  3. 使用npm安装国内镜像 npm install -g cnpm --registry=https://registry.npm.taobao.org
  4. 使用cnpm安装hexo cnpm install -g hexo-cli
  5. 创建一个blog文件夹来存放你的博客内容mkdir blog
  6. 初始化并生成你的博客 sudo hexo init 查看自动生成的博客hexo s
  7. 接下来部署到你的github，新建一个库，注意名字:
  8. 在blog文件里安装一个git插件方便部署 cnpm install --save hexo-deployer-git
  9. 修改 _congfig.yml中的最下面的Deployment，type写git，增加repo写刚刚创建的仓库地址，branch写master分支，不写也行。注意冒号后面的空格
  10. 接下来就部署到远端：hexo d 输入github的账号密码，接下来就可以访问你的git地址查看了。
  11. 写新博客hexo n "blogname"
  12. 重置一下博客hexo clean
  13. 重新生成静态页面hexo g
  14. 本地运行查看hexo s
  15. 部署到服务器 hexo d
  16. 如果要更换主题还是在_config.yml中找到theme改为你想换的主题就好了，不过前提要先把主题下载到themes/name中。
