2019-07-17 Comments

python学习笔记

字符类型

  1. 字符类型：python也有整数型，字符串型，浮点数，布尔型（用and，or，not运算），空值（None）。
  2. 输出语句：print 'hello, world'
  3. python的注释是#或# #
  4. 在Python程序中，变量是用一个变量名表示，变量名必须是大小写英文、数字和下划线（_）的组合，且不能用数字开头.
  5. 字符串可以用''或者""括起来表示。如果字符串本身包含'怎么办？比如我们要表示字符串 I’m OK ，这时，可以用""括起来表示，转义字符也是\.
  6. python中整数和浮点数运算是浮点数。

    list有序列表

  7. Python是动态语言，所以list中包含的元素并不要求都是同一种数据类型，我们完全可以在list中包含各种数据：L = ['Michael', 100, True]
  8. 访问list数组第一个元素L[0],访问最后一个元素L[-1]。
  9. list中添加元素有append（element）和insert（num，element）两种,前一种添加到尾部。
  10. list中删除元素：pop()不填写元素直接删除最后一个，填写下标删除对应的元素。
  11. L[1]="a"直接替换角标为一的元素为a.

    tuple另外一种有序列表，叫做元祖，和list类似，但一旦创建不能修改

  12. 创建一个tuplet=(1,2,3,4,5)，其他方法和list相同。创建0个元素的tuple是t=(),但创建单元素的tuple时候，要加逗号t=（1，）

    条件判断和循环

  13. 注意:Python代码的缩进规则。具有相同缩进的代码被视为代码块，上面的3，4行print语句就构成一个代码块（但不包括第5行的print）。如果 if 语句判断为True，就会执行这个代码块。缩进请严格按照Python的习惯写法：4个空格，不要使用Tab，更不要混合Tab和空格，否则很容易造成因为缩进引起的语法错误。注意: if 语句后接表达式，然后用:表示代码块开始。如果你在Python交互环境下敲代码，还要特别留意缩进，并且退出缩进需要多敲一行回车：
  14. python中有if…elif…else…
  15. python中的for循环和while循环,break跳出当前循环，continue跳过后续代码继续循环。

    Dict和set类型

  16. dict相当于java当中的set，有键值对，len（d）来表示集合的大小,dict是无序的，而且key必须要不能改变的值来作为key，而且key不能重复。

    迭代：Python中，迭代永远是取出元素本身，而非元素的索引。

高阶函数

高阶函数就是把函数作为参数传递到另一个函数中。

  1. map()函数，可以传入两个参数，第一个为函数，第二个为函数参数list
  2. reduce()和map()差不多，但是函数必须有两个参数，而且是会重复调用函数f，reduce()还可以接收第3个可选参数，作为计算的初始值。
  3. filter()函数是 Python 内置的另一个有用的高阶函数，filter()函数接收一个函数 f 和一个list，这个函数 f 的作用是对每个元素进行判断，返回 True或 False，filter()根据判断结果自动过滤掉不符合条件的元素，返回由符合条件元素组成的新list。
  4. python函数可以返回函数。
  5. 像内层函数引用了外层函数的变量（参数也算变量），然后返回内层函数的情况，称为闭包（Closure）,返回值函数不能引用任何后续可能发生变化的量。
  6. 匿名函数lambda：关键字lambda 表示匿名函数，冒号前面的 x 表示函数参数。
