什么是pojo？

1。POJO(Plain Ordinary Java Object)是简单的java对象，POJO的内在含义是指那些没有从任何类继承、也没有实现任何接口，更没有被其它框架侵入的java对象。
2。具有一部分getter/setter方法的那种类就可以称作POJO，而JavaBean要复杂的多。
3。JavaBean是可复用的组件，对JavaBean并没有严格的规范，理论上任何一个Java类都可以是一个Bean，pojo的格式用于数据的临时传递，它只能装在数据，作为数据存储的载体，而不具有业务逻辑处理的能力。而javabean虽然数据的获取与pojo一样，但是javabean当中可以有其它的方法。a.这个类必须有一个公共的缺省构造函数。

b.这个类的属性使用getter和setter来访问，其他方法遵从标准命名规范。

c.这个类应是可序列化的。
